# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in jsLPSolver, please report it responsibly by emailing:

**justin.w.wolcott@gmail.com**

Please include:
- A description of the vulnerability
- Steps to reproduce the issue
- Potential impact
- Any suggested fixes (optional)

## Response Timeline

- **Acknowledgment**: Within 48 hours
- **Initial assessment**: Within 1 week
- **Resolution target**: Depends on severity, typically within 30 days

## Scope

This policy applies to the jsLPSolver npm package (`javascript-lp-solver`) and its source repository.

## Supported Versions

Security updates are provided for the latest major version only.

| Version | Supported          |
| ------- | ------------------ |
| 0.4.x   | :white_check_mark: |
| < 0.4   | :x:                |
